# 🎨 点点组件库

欢迎使用点点组件库！这是一个基于 **Svelte 4** 构建的现代化组件库，提供了丰富的 UI 组件和优秀的开发体验。

## ✨ 特性

- 🚀 **现代化** - 基于 Svelte 4 最新特性
- 🎨 **美观** - 精心设计的视觉风格和动画效果
- 📱 **响应式** - 完美适配各种屏幕尺寸
- ♿ **可访问性** - 遵循 WCAG 标准，支持键盘导航
- 🔧 **可定制** - 丰富的配置选项和主题支持
- 📚 **文档完善** - 详细的组件文档和使用示例

## 🚀 快速开始

### 安装

```bash
npm install @taole/chatroom
```

### 使用

```javascript
import { Button, Card, Input } from '@taole/chatroom';

// 在你的 Svelte 组件中使用
<Button variant="primary" on:click={handleClick}>
  Click me
</Button>
```

## 🎯 设计原则

### 一致性
所有组件都遵循统一的设计语言，确保视觉和交互的一致性。

### 可访问性
每个组件都经过精心设计，支持屏幕阅读器和键盘导航。

### 性能
轻量级实现，最小化运行时开销，提供流畅的用户体验。

### 可扩展性
组件设计考虑了各种使用场景，支持灵活的自定义和扩展。

## 🎨 主题色彩

我们的组件库使用了一套精心设计的色彩系统：

- **主色**: `#667eea` - 用于主要操作和强调
- **辅助色**: `#764ba2` - 用于次要操作和装饰
- **成功色**: `#10b981` - 用于成功状态和确认操作
- **危险色**: `#ef4444` - 用于错误状态和危险操作
- **警告色**: `#f59e0b` - 用于警告状态和提醒
- **信息色**: `#3b82f6` - 用于信息展示和提示
